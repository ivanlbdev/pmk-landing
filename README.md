# ПМК

## Сборка и запуск
> Собиралось на `Node.js` версии `^14`.

Устанока зависимостей:
```shell
$ cd .pmk/
$ npm install
```

Запуск сервера для разработки с горячей перезагрузкой:
```shell
$ cd .pmk/
$ npm run start
```

Сборка в dev и prod режиме:
```shell
$ cd .pmk/
$ npm run dev
$ npm run build
```
